Git Commands

General Commands

  • git config --global user.name "USERNAME" : configure username globally
  • git config --global user.email "USEREmail" : configure user email globally
  • git config --global color.ui true : show outputs colorfully
  • git init: initialize Git in the current directory
  • git status: show modified, untracked, and staged files
  • git add: add untracked or modified files to the stage
  • git add . & git add -A: add all untracked or modified files to the stage
  • git commit: commit staged files and remove them from the stage
  • git commit -m "Title" -m "message" : commit with title and message
  • git log: show a log of commits with date, descriptions, and author name
  • git log --one-line --graph --all --decorate: show logs like a tree with all branches
  • git diff <...>: compare two versions of a file
    • git diff HEAD: compare last committed state vs current state for all modified files
    • git diff --staged: compare last committed state of staged files vs current state
  • git reset filename: unstage a file
  • git reset commitHash: go back to a specified commit (modifications will be unstaged)
  • git reset commitHash --hard: fully return to a specified commit and remove modifications
  • git checkout -- filename: discard last modifications in a file and return to the last committed version
  • git rm filename: remove a file from both Git and the system

Branches

  • git branch: show list of all branches
  • git branch branchName: create a new branch
  • git branch -d branchName: remove a branch (with committed changes)
  • git branch -D branchName: remove a branch (even without committed changes)
  • git checkout branchName: switch to another branch
  • git merge branchName: merge another branch into the current branch

Remote & Clone

  • git clone gitAddress: clone a Git repository
  • git remote add gitAddress: add a remote repository (another address)
  • git remote -v: list all remote addresses
  • git push origin master: push the local master branch to the remote repository
  • git pull origin master: pull updates from the remote master branch to the local repository

Tags

  • git tag: list all tags
  • git tag -a <...> -m '' : create a tag
  • git tag -a <...> commitHash -m '' : create a tag for a specific commit
  • git tag -v <...> verifytag -v: verify a tag
  • git show tagName: show details of a tag
  • git push origin tagName: push a specific tag to the remote repository
  • git push origin --tags: push all tags to the remote repository
  • git blame: show who last modified each line in a file
  • pgp (gpg in open source world): used for signing and verifying tags

Bisect

  • git bisect: used for debugging by binary search
